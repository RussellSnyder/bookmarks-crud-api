# Bookmark Api

Basic CRUD application built with NestJs.

Features:
- Authentication
- User CRUD
- Bookmark CRUD
- Postgres docker
- e2e tests

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# e2e tests
$ npm run test:e2e
```
